# 📓 Caderno Digital

Seu guia de estudos em Python, sempre à mão.

## 💡 Sobre o Projeto

Criado para resolver um problema pessoal: dificuldade em lembrar 
conteúdos estudados sem precisar abrir cadernos físicos ou procurar 
em vários arquivos. O Caderno Digital centraliza tudo em um só lugar.

## 🚀 Versões

- ✅ **V1** — Lógica pura em Python no terminal (POO, listas, aspas triplas)
- ✅ **V2** — Interface visual com Streamlit
- 🔜 **V3** — Banco de dados com SQLite
- 🔜 **V4** — API com FastAPI
- 🔜 **V5** — Polimento e funcionalidades extras

## 🛠️ Tecnologias

- Python
- Streamlit

## ▶️ Como rodar

```bash
pip install streamlit
streamlit run V1_do_meu_projeto.py
```
